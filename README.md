# Convolutional-Neural-Networks-Cats-and-Dogs
Этот репозиторий представляет собой простую реализацию сверточной нейронной сети (CNN), в которой используется предварительно обученный слой MobileNet. Для сокращения затрат времени и ресурсов применён механизм ранней остановки обучения (EarlyStopping).
Точность контрольной выборки: 0.97
